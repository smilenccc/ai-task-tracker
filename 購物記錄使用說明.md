## 🚀 使用方式（三種）

### 方式 1：傳截圖給我（推薦）📸

**步驟：**
1. 在 Coupang 完成購買後截圖訂單頁面
2. 傳給我（Telegram）
3. 我會自動解析並詢問確認
4. 確認後自動記錄到系統

**範例對話：**
```
你：[傳截圖]
我：看到了！您購買的是：
    • Nintendo 任天堂... - $13,850
    • 國泰世華信用卡優惠 - $10,199
    確認記錄嗎？
你：確認
我：✅ 已記錄！
```

---

### 方式 2：直接告訴我 💬

**格式：**
```
Linda，我在 Coupang 買了 [商品名稱]，花了 [金額] 元
```

**範例：**
```
Linda，我在 Coupang 買了 Nintendo Switch，花了 13,850 元
```

我會自動記錄並回覆確認。

---

### 方式 3：詳細記錄（可選） 📝

如果你想記錄更詳細，可以這樣說：

```
Linda，記錄購物：
商品：Nintendo Switch
```

## ❓ 常見問題

**Q: 如果買了多個商品怎麼辦？**
A: 可以分開記錄，或合併成一筆（看你習慣）

**Q: 可以修改或刪除記錄嗎？**
A: 可以！告訴我要改哪一筆，我會幫你處理

**Q: 可以追蹤其他網站嗎？**
A: 當然！任何購物網站都可以（蝦皮、PChome、momo 等）

**Q: 資料會遺失嗎？**
A: 不會！所有資料都存在 GitHub，永久保存
